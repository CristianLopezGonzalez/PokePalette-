# PokePalette-
PokéPalette es una aplicación web que une el mundo Pokémon con el diseño gráfico. El usuario busca cualquier Pokémon, y la app extrae automáticamente los tres colores predominantes de su sprite oficial, generando una paleta lista para usar en cualquier proyecto creativo. Los usuarios pueden guardar paletas, crear colecciones y explorar las de otros
